# Attendance Management System (Console-based) - Skeleton

## Quick start (Mac, VS Code)
1. Install Java 11+ and ensure `java` and `javac` are in PATH.
2. Open this folder in VS Code.
3. Compile: `javac -d out src/**/*.java`
4. Run: `java -cp out main.Main`

## Notes
- Subjects taken from your PDF header are included by default in `Main.java`.
- Threshold = 75% (monthly).
- Months for demo: August, September, October (use YearMonth like `2025-08`).
- Alerts are simulated and saved to `resources/alerts.txt` when generated.
